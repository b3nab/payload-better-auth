# Quick Start Guide

This guide will help you get started with Payload Better Auth quickly and easily.

## Prerequisites

- Node.js 18.20.2 or higher
- Payload CMS project
- pnpm (recommended) or npm/yarn

## Installation

1. Install the package:

```bash
pnpm add payload-better-auth
```

2. Generate a secure secret:

```bash
openssl rand -base64 32
```

3. Add the secret to your environment variables:

```env
BETTER_AUTH_SECRET=your-generated-secret
```

## Basic Setup

1. Import and configure the plugin in your Payload config:

```typescript
import { buildConfig } from 'payload'
import betterAuthPlugin from 'payload-better-auth'

export default buildConfig({
  plugins: [
    betterAuthPlugin({
      betterAuth: {
        appName: 'My App',
        secret: process.env.BETTER_AUTH_SECRET,
      },
    }),
  ],
})
```

2. Start your Payload server:

```bash
pnpm payload dev
```
