## Quick Start

```bash
npm install payload-better-auth
# or
yarn add payload-better-auth
# or
pnpm add payload-better-auth
```

## Basic Usage

```typescript
import { buildConfig } from 'payload'
import betterAuthPlugin from 'payload-better-auth'

export default buildConfig({
  plugins: [
    betterAuthPlugin({
      betterAuth: {
        appName: 'My App',
        secret: process.env.BETTER_AUTH_SECRET,
      },
    }),
  ],
})
```

## Why Payload Better Auth?

- **Enhanced Security**: Built-in two-factor authentication and advanced security features
- **Seamless Integration**: Works out of the box with Payload CMS
- **Flexible Configuration**: Easy to customize and extend
- **Type-Safe**: Full TypeScript support
- **Modern Architecture**: Built with modern best practices
